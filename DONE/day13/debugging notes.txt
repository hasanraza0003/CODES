DAY 13
DEBUGGING

THE STEPS INCLUDE IN DEBUGGING IS GIVEN BELOW :

1. DESCRIBE THE PROBLEM
2. REPRODUCE THE BUG
3. PLAY WITH COMPUTER  (RUN)
4. FIX THE ERRORS
5. PRINT IS YOUR FRIEND  (PRINT ALL THE OUTPUTS OF THE CODE FOR ANALYZE THE PROBLEM)
6. USE A DEBUGGER (https://pythontutor.com/visualize.html#mode=edit)
7. TAKE A BREAK FOR SOMETIME.....
8. ASK A FRIEND OR GOOGLE IT THAT PROBLEM
9. RUN OFTEN (AFTER CHANGING A SINGLE STEP , RUN THE PROGRAM )
10. IN THE END NO WAY TO SOLVE THE PROBLEM  ---   STACK OVERFLOW
